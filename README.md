<h3> Installation </h3>

<h3> Front End</h3>

<p> First install node modules in the command line:</p>

```sh
npm install
```

<p>Run the front end with npm:</p>

```sh
npm start
```
<h3>Back End</h3>

<p> Install node modules just like front end:</p>

```sh
npm install
```
<p>Run the server using npm:</p>

```sh
npm run dev
```
